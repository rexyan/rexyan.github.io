安装 Ollama

https://ollama.com/ 支持了 Windows，Mac 和 Linux 下的安装包，下载后直接运行即可安装！

运行 Llama3

使用 ollama pull llama3 拉取 llama3 的模型，使用 ollama run llama3 运行模型。

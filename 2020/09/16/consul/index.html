Python 使用

参考 github 上的项目进行了简单的修改，修改后地址为 新地址，使用方式：

  1. 启动 consul
  2. 将 consul 地址配置到代码中
  3. 启动项目（注意修改启动 IP 为本机 IP，因为 consul 会对项目做健康检查，检查不通过无法调用服务）
  4. 查看 consul web 页面，查看服务是否正常
  5. 执行 aaaa.py 脚本，调用 consul 上已经注册的服务

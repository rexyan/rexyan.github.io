代码跳转

右上角Open Perspective 添加Pydev
将项目所在文件夹加入到System PYTHONPATH(windows -> Preferences -> Pyenv -> Interpreters -> Python interpreters -> libraries -> New Folder)

为Web项目添加运行端口

选择main.py -> Run Configurations -> Arguments -> 添加端口8088，这样就能为项目设置运行参数

快捷键

  1. 查看当前所有的方法：ctrl + o
  2. 快速切换编辑的文件：ctrl + e
  3. 上下移动一行： alt + 上下方向
  4. 前进、后退： alt + 左/右
  5. 下面新建一行： 回车/shift + 回车
  6. 上面新建一行：ctrl + shift + 回车
  7. 快速搜索一个文件：ctrl + shift + r
  8. 搜索：ctrl + h
  9. 快速复制一行：ctrl + alt + 方向
  10. 快速删除一行：ctrl + d
  11. 转大写 ctrl + shift + x
  12. 转小写 ctrl + shift + y

安装

安装地址

浏览器中输入chrome://apps/进入谷歌网上应用商店

问题

如果进不去则需要翻墙，如何翻墙，请看下面

翻墙

翻墙需要下一个代理软件，并且需要代理服务器。没关系，都准备好了

下载软件

https://github.com/shadowsocks/shadowsocks-windows/releases/download/4.0.10/Shadowsocks-4.0.10.zip

解压安装


解压后得到一个文件，直接双击打开就好，如若不能打开按照提示升级.NET版本就好

填写帐号

打开软件后，右下角会出现一个小飞机的图标，双击打开面板
填写服务器地址为：104.224.151.119
端口为：9993
密码为：”你的名字的全拼” (真的是姓名的全拼，不是这里的汉字)
选择加密方式为：aes-256-cfb
其他的不用管，直接点击确定

开启

右键点击小飞机，选中第一项，启动系统代理（开关），开机启动可选（是否开机启动）

更新PAC

选择PAC项目，然后选择从GFTList更新本地PAC，更新成功后会有弹窗提醒！

实现访问国外网站

接下来便可以愉快的访问国外的所有网站了
谷歌 https://www.google.com/
youtube https://www.youtube.com/
…..
只要是被强的都能访问

PAC模式和全局模式

PAC模式： 只有不能访问的网站（被墙的网站），才会使用该软件去代理访问（平常使用选择PAC模式即可，不然如果开启全局模式的话， 访问百度你会从国外绕一圈再回到国内）
全局模式： 不管能不能打开的网站，都是用该软件（不推荐长期开启此模式，因为网络上的数据包会从国外【代理的服务器在洛杉矶，数据就会去洛杉矶，然后在回到你的电脑上】绕一圈再回来）

翻墙的原理是什么

原理就是你不能访问的东西，代理服务器替你去访问，然后再将数据返回给你。

插件

搜索安装

直接打开第一步进入谷歌网上应用商店，在红色部分搜索安装插件了，安装插件后，插件会显示在浏览器数据框的后面

推荐插件

  1. 划词翻译（选中文本后出现”译”，点击一下就能翻译了，什么有道翻译之类的就可以卸载了）

  2. LastPass（一个网页密码记住的插件，很多网站注册的时候填写密码，后来就忘记了，这个插件可以帮助你记住密码，下次来直接登录）

  3. tampermonkey （又叫做油猴，很强大很强大的一个插件，你是否遇到过百度云下载大文件很慢，且要打开百度云客户端？追剧有广告，还要会员？等等等等，有了油猴，这写都轻松解决，安装油猴后，去油猴论坛找自己喜欢的脚本，安装即可，地址：https://greasyfork.org/zh-CN）

  4. AdBlock（广告拦截插件，网页上的广告统统都没了）

  5. 新浪微博图床（图床就是存储图片的网站，只需将图片上传，你就能得到一个该图片的链接，然后就可以到处分享和粘贴了，这篇文章的图片就是上传到新浪图床上的）

  6. 更多可以看这里的推荐（https://guanguans.cn/2018/03/25/Extended/ ) 当然也可以直接去谷歌网上应用商店寻找

第二步比较复杂，但是非常值得的，因为以后都可以翻墙了，手机上也可以，家里电脑，公司电脑都可以。windows需要下载上面介绍的软件，手机（iphone）需要下载Wingy。同样的填写帐号密码就能实现翻墙了。实现翻墙后最好去注册一个谷歌帐号，然后在浏览器里面登录该账号，这样家里，公司，手机等等等，的保存的网站标签啊，访问历史记录啊，都能同步了。

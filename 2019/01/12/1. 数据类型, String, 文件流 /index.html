二, 引用数据类型 String 中的方法(4532)

第一组：判断方法(重要的为4个)

boolean equals(String str);            // 比较两个字符串的内容是否相等
boolean equalsIgnoreCase(String str); // 比较两个字符串的内容是否相等，忽略大小写
boolean startWith(String subStr);       // 判断某个字符串是否以指定的子串开头
boolean endsWith(String subStr);       // 判断某个字符串是否以指定的子串结尾

第二组：获取方法(重要的为5个)

int length();                       // 获取字符串中字符个数
char charAt(int index);           // 获取字符串中的某一个字符
String subString(int startIndex); //从指定下标开始截取字符串，直到字符串的末尾
String subString(int startIndex, int endIndex); //从指定下标开始截取字符串，到指定下标结束(截取结果为左闭右开)
int indexof(String subStr);       // 获取子串第一次出现的下标

第三组：转换方法(重要的为3个)

String toLowerCase(); // 转成小写串
String toUpperCase(); // 转成大写串
Char[] toCharArray(); // 变成字符数组

第四组：其他方法(重要的为2个)

String trim();                 // 去掉字符串两端的空格
String[] spilt(String str); // 切割字符串

三, 流(读写文件)

输出流：数据从 java 程序到文件中

FileWriter: 文件的字符输出流 (写数据)

package me.yanrs.demo01;
import java.io.FileWriter;
import java.io.IOException;
public class HelloWorld {
    public static void main(String[] args) throws IOException {
        FileWriter fWriter = new FileWriter("1.txt");
        char[] charArray = {'a', 'b'};

        fWriter.write('a');             // 写入字符
        fWriter.write(100);              // 写入字符的 Ascii 值
        fWriter.write(charArray);        // 写入字符数组
        fWriter.write(charArray, 0, 2); // 写入字符数组的一部分，0为开始位置，2为长度

        fWriter.write("12345");            // 写入字符串
        fWriter.write("java",1, 2);     // 写入字符串数组的一部分，1开始，2为长度

        fWriter.close();
    }
}

输入流：从文件中到 java 程序

FileReader: 文件的字符输入流

package me.yanrs.demo01;
import java.io.FileReader;
import java.io.IOException;
public class HelloWorld {

    public static void main(String[] args) throws IOException {
        FileReader fileReader = new FileReader("1.txt");
        int ch = fileReader.read();             // 读取一个字符，返回 ascii
        System.out.println((char) ch);            // (char) ch 将 ascii 转为字符

        char[] charArray = new char[5];
        int len = fileReader.read(charArray);   // 以字符数组的方式读取文件的内容，charArray 的长度是读取内容的范围大小, 返回值是真实读取到的大小
        System.out.println(len);
        System.out.println(charArray);            // 输出读取到的内容

        fileReader.close();
    }
}

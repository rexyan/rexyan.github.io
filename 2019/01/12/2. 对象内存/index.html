一个标准的类，应该包含有：

  1. 封装(private + get/set)
  2. 构造(有参或者无参)
    `java
    package me.yanrs.demo2;

public class Dog {
private int age;
private String name;
// Mac 中生成get set 方法的快捷为 option + command + s -> Generate Getters and Setters 或者 右键 Source -> Generate Getters and Setters

// 构造函数
// 1. 无参构造 Generate Constructors from Superclass 
public Dog() {
}

// 2. 有参构造 Generate Constructors using Fields
public Dog(int age, String name) {
    super();
    this.age = age;
    this.name = name;
}

// get set 方法
public int getAge() {
    return age;
}
public void setAge(int age) {
    this.age = age;
}
public String getName() {
    return name;
}
public void setName(String name) {
    this.name = name;
}

}
`

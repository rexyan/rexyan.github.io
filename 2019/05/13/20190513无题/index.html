这几天买了几本书，这是今年的实体书目标，希望可以看完。

这几天在做 Java 项目，有很多地方都觉得生疏，不过做完之后应该有很多可以总结的地方，自己也可以做一个单独的项目来练练手，结合 React 或者 Vue。接下来要把 K8s 学习完成，把日语进度追赶上，然后就是提高 Java 和 Python 的水平，Python 3.8 马上就出来了，到时候出一个学习的详细计划。以上目标我觉得今年可以完成就已经非常的棒了。我也相信不出意外的话是可以的。

生活方面，需要慢慢放下格格。应该节约用钱，多存一些，年初定的目标应该是没问题并完全可以完成的，现在自己有了更大的想法，想多存一些。现在自己将公积金做调整后我感觉也是可以达到的，就希望自己不要乱花就行。理财方面，还是以稳定的定期为主，基金和 P2P 先不碰。

国庆可能将老爸老妈接过来玩几天，不过目前也只是想法而已，况且老爸现在在学驾照。我现在也没有学，原因是不可能在上海买车，离家太远了，根本不可能开会去。所以自己学驾照的事情估计今年也没有什么太大的进展。宠物方面，自己一直在反思，是不是自己的教育方法有问题，也一直在考虑要不要再重新养一只猫，毕竟各种东西都在，现在还有几袋猫粮和猫砂。

还有晚上要坚持以前的手机不上床的原则，尽量晚上十一点之前睡觉。

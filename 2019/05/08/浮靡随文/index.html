浮靡随文

发表于 2019-05-08 | 分类于 博客 | 本文总阅读量 次

很长时间没有写类似的小记了，大多都是一些技术的文章。今天想法多了些，就想记录下来。这两天在看《人生海海》，还不错的一本书，每天晚上睡觉前或者上班中午不睡午觉都会看一下。日语马上就要考试了，也没有学习，结果肯定不言而喻了！我也相信和体会过做一件事开头都比较难，其次是坚持，什么事都会有一个结果，有时只是少了一颗敢于直面的心。工作上马上开始了一个陌生的领域，开始做一些 Java 的项目，这一点对我来说还是挺有吸引力，因为不管是对个人发展还是工作所需，我都愿意去尝试和接触。 没人知道你到底付出了多少，也没人知道我 Java 曾经大起大落，也许只看到的是你表现出来的那部分，而最直观和最直接体现个人好坏的就是这一部分，有时候就在想，别人的看法就那么重要吗？我现在的答案是，如果我认为你是一个值得我肯定的人，那么你对我的看法就很重要，否则就是被我舍弃的那一部分。每天都在笑，但又有多少笑是真的呢！

生而敏感的人，也许真正的快乐，都不容易浮现！浮浮沉沉，走走散散，有新的来，有旧的走。我不敢想以后，也不敢回头！最近也迷上了古风音乐，前不久看了一本《显微镜下的大明》，端午打算到南京走走，出去散散心，整理一下心情 —— 没有什么是永恒的。

我喜欢格格，尽管是我死气白脸的赖着别人，人生中又有什么是十全十美的呢？我甚至陷入了一种不能自拔的地步，我怕最后自己遍体鳞伤。喜欢她的知性，成熟，内涵，正如她的签名，“浅水作响，静水流深！”，这个道理在初中的都时候就知道了，但自己却一直没有办法做到，也许在别人眼里我很优秀，包括我的朋友，我的亲戚。也有很多人有认为我很幼稚，但又有谁知道我真实的面目呢？

你如三月的春风，带来了一缕温暖的阳光，我躺在草地上，望着天空，无忧无虑。我不相信宗教，但自己却认同和轮回相同概念的 “圆”。正如我也相信与阴阳相同概念的 “正反”。

睡觉都会做梦，这好像成了我一直不变的习惯。每天早上醒来，都会大惊。恍如隔世，犹如翻滚。有时候我都觉得自己有抑郁症。我认同自己是一个矛盾体，有些时候也不知道自己在做什么，到底还是感情动物。在 “ 多抓鱼 ” 上上面用小号买了两本书，《了不起的盖茨比》和《霸王别姬》，小时候经常吧 “了不起的盖茨比” 想成是一本专门些比尔盖茨的书，直到后来才知道这是一个屌丝逆袭，最终失败的故事！“霸王别姬” 讲的可不是楚霸王和虞姬的故事，而是一本让我震撼的作品。我喜欢格格，书就是这其中很重要的一个原因。书让人有了灵魂，有了想法，我想格格懂的道理，读过的书也许比我多，正如她比我大一样。蜀道难，何尝现在自己不难？

晚安，格格。晚安，格格的哥。

尝试下 bee 命令，若没有，则需要将 bee 加入环境变量。

在 go 的 src 下面新建 rexyan 文件夹，并在终端中切换到该文件夹，执行以下命令创建一个 beego 的项目，名称为 blog。

bee new blog

进入 blog 目录，启动项目

bee run

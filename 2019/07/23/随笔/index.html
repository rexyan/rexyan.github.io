阅读时长:   5 分

博客好像有几天没有更新了。自己这几天一直在调整自己的状态。也许是天气热了，稍微有点浮躁。不过最近这段时间都有学习。因为我感觉时间不够用，所以给自己定了计划，并且由以前的月计划缩小为周计划，这样灵活性较高，可调整控性强，自己的收获感也高，比较容易激发起兴趣。

最近看完了《假面山庄》《远山淡影》。不愧是非常有名的作品。关于这两本书，我想可以完全用完整的一篇文章来描述了。技术学习方面，自己的 Python 相对于之前稍微有点提升，但是我认为不够，且和自己的工作年限不符合，这方面还需要加强，Java 嘛有提升，不大，自己做一个项目应该没有多大问题，无非就是熟练和优化的问题，但我认为还是停留在表面。那我这段时间都在干什么了呢！！这段时间看了 ES，因为工作中会用到，但是自己认为也才是入门阶段，很多高级用法，原理，还需要继续深入才行。针对于 K8s，我认为自己到了一定瓶颈了，不是自吹，我认为自己是公司最掌握的人之一。但是还是很浅，很多高级用法，配置，组件还不会使用，比如有状态的 Pod，存储部分等等。日语部分，这玩意差不多有两年了，进度真的很差，三天打鱼，两天晒网，但是自己又没有放弃，导致分分合合，纠缠不清。于是在周计划里面，加重了它的分量，今年12月一定要考 N3，我一直在怀疑和反思是不是自己学习计划有点问题，回头去看，的确有一些，希望能及时止损，修改出一个适合自己这种比较笨的人的计划。健身方面，买了蛋白粉，自己也每天都在坚持锻炼，前段时间全身疼，手臂都打不直，不过效果是有的，能感觉到体型的微变化，肌肉的变化。我想说的是不光每一个胖子都是潜力股，瘦子也是！！金钱方面去年定下的目标已经可以实现，且会超。爸妈可能国庆会来上海，但是好好的陪他们玩几天，国庆前期自己应该会去云栖大会，如果他们不来的话，我估计就直接从杭州回家了。后续有什么计划呢？那就是按照自己的计划进行，不慌不忙，坚持下去，一定比浮躁的学习，或者没有计划的学习要强很多，也算是磨一磨自己的性子吧。

前公司搬到了普天产业园，格格肯定也在。这事怎么说呢？过！提起来就觉得乱，和自己有关系，和格格没关系。我发现程序员是真的木啊，都说 “男人的嘴，骗人的鬼” 我倒是想骗，但是口才不允许，今年中秋整满二十四，二十五，六是时候开始找对象了。格格怎么办？？我想会渐渐淡忘吧，别人的确很优秀，我和她我想年龄是最大的障碍吧！不过说实话挺害怕结婚恋爱的！我是怕以后，中年人的婚姻生活。分房睡，带孩子，房贷，车贷….我想逃！好好练肌肉，然后找一个优秀，三观对得上，而且很好的对象，哈哈哈哈！

最后放上我的计划，当当当！

  1. 每天早上七点起床
  2. 七点到八点背书
  3. 八点二十出门（头发每天都爆炸💥，不过不管怎么炸都没有重新洗，先不管。另外我烫发或者把头发稍微吹蓬松一点挺好看的，自己认为。因为现在都没管，算了，整了也没人看）
  4. 每天来回地铁看书（以前是背单词，后来发现很吵，而且很挤。还有很对人背一个大包！！！不知道包要放下来吗！我一般背包，不管大或者小，都是提在手上的，最差也是背在前面。）
  5. 中午看书，现在争取不午休了。但这就要保证晚上的睡眠充足。
  6. 晚上回来，一三五学日语，二四技术。周六日因为不上班，所以肯定比平时学习的多，所以不做安排。
  7. 写日记，哪怕是一句话都要写！别人写日记都像是写小说，而我，额～，从小就养成了记流水账的习惯。
  8. 每周出去玩一次，为什么设置这个呢？因为连我自己都觉得自己宅，这样怎么找女朋友！！！！哪怕是花钱买门票，没事，去！只要不是情侣比较容易吃狗粮的地方，不过话说回来现在哪儿是适合单身而没有情侣的呢。很想去图书馆，有空调，还安静，可以安安静静的看书，学习，写博客。而且图书馆爱学习，提升自己的妹子肯定都不差！
  9. 记录开销，时刻提醒自己不要乱花钱，每月复盘一次。

最后如果自己做得不错的话，会给自己一些奖励，比如这周的就是 “吃一次烧烤” 。虽然吃一次烧烤没啥，也花不了多少钱，但是我觉得这样非常有意义。这也是自己为什么设置周计划而不是月计划的原因，因为周比较容易执行和达到，这样的话，一个月就有四次奖励，而月计划的话才一次，哈哈哈哈😂。仿佛自己在骗自己，多重人格，成熟的照顾幼小的。

先写这么些，啥时候记起来了再写，健身完开始写的，还没有洗澡，得马上洗澡，睡觉，不然明天起不来！不得不说，万事开头难。
